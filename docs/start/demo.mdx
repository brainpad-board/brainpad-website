# Online Demo

---

![DUELink Demo website](pathname:///img/demo-website.png)

This [Online Demo](https://demo.duelink.com/) uses [JavaScript](../language/javascript) through a web browser to run some of the DUELink [Standard Library](../engine/stdlib) commands.

Plug in directly boards that have a [USB](../interface/usb) connector, or use [USB Hook](../catalog/accessory/usb-hook) to connect any other module.
