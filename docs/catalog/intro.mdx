# Product Catalog

DUELink products include a verity of Sensors and Actuators, and Microcomputers. Every single one of them can work with several [Supported Systems](../system/intro) using one of [Hosted Languages](../language/intro). They can also run [Non-hosted](../language/nonhosted) and [Standalone](../engine/standalone) using the [Internal Engine](../engine/intro).

:::tip
Each modules has a unique ID that allows for dynamic discovery and use, shown in summary table below. The `Version()` command returns the product ID.
:::
