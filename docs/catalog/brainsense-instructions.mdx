# BrainSense Assembly Instructions

Start with the front side, get the wooden front side and the four sensors:

- RGB LEDs
- Light Sensor
- Motion Sensor
- Temperature & Humidity Sensor

Two screws and nuts for each sensor. Make sure to mount screws like the images attached.

To finish this smart home, you need to add the automatic door. Bring the front side and:

- Servo motor
- Small wooden part
- Two small screws and nuts

![BrainSense inside view of front panel with fan](pathname:///img/catalog/brainsense_assembly/bsa_20.png)

Add the wooden piece and then the motor, then connect all sensors with wires:

<div style={{ display: 'flex', justifyContent: 'center', gap: '1rem' }}>
  <img src="/img/catalog/brainsense_assembly/bsa_21.png" alt="BrainSense front panel inside view" width="500" />
  <img src="/img/catalog/brainsense_assembly/bsa_22.png" alt="BrainSense front panel outside view" width="500" />
</div>

Fit the wires inside the spaces up each sensor, and connect it with the controller board, don’t forget the two sides sensors

Make sure to connect the sensors as follows:

- Servo PIN 10	
- Fan PIN 11	
- RGB LEDs PIN 12	
- Horn PIN 13	
- PIR   PIN 3
- Flame PIN 4	
- STEAM PIN 5	
- Humid PIN 6	
- Light PIN 0	
- Soil  PIN 1

![BrainSense wiring](pathname:///img/catalog/brainsense_assembly/bsa_23.png)

Connect the soil sensor from the small circuit in the home back.

<div style={{ display: 'flex', justifyContent: 'center', gap: '1rem' }}>
  <img src="/img/catalog/brainsense_assembly/bsa_24.png" alt="BrainSense with soil sensor not attached" width="500" />
  <img src="/img/catalog/brainsense_assembly/bsa_25.png" alt="BrainSense with soil sensor attached" width="500" />
</div>

Then assemble the front and the rest of the home using the wooden lock.

![BrainSense assembled with wooden locks](pathname:///img/catalog/brainsense_assembly/bsa_26.png)

To Install the door, we need the wooden door, one small screw, and a servo adapter.

Connect the servo adapter with the servo motor, and then put the wooden door and stick it with the screw.
