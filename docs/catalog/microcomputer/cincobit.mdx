<TabItem value="drivers">

| Function    | Description                                               |
| ----------- | --------------------------------------------------------- |
| `Init()`    | Initialize the display. Automatically called on power up. |
| `GetW()`    | Get display width: 5                                      |
| `GetH()`    | Get display height: 5                                     |
| `IsColor()` | Returns 0.                                                |
| `BtnUpL()` | Returns 1 if button left was released.                                                |
| `BtnDownL()` | Returns 1 if button left was pressed.                                                |
| `BtnUpR()` | Returns 1 if button right was released.                                                |
| `BtnDownR()` | Returns 1 if button right was pressed.                                                |
| `PlayBeep(f,d)` | Play beep. f: frequency, d: duration                                               |
| `Light()` | Returns volt read.                                                |
| `DVer()`    | This driver version.                                      |

:::tip
See [Graphics](../../engine/graphics) for available drawing functions.
:::

    TextT("DUELink", 1, _x, 0)
    Show()
    _x= _x-1

from DUELink.DUELinkController import DUELinkController
import time
availablePort = DUELinkController.GetConnectionPort()
duelink = DUELinkController(availablePort)
x = 0
while True:
    duelink.Graphics.TextT("DUELink", 1, x, 0)
    duelink.Graphics.Show()
    x -= 1
    if x < -40:
        x = 6
    time.sleep(0.1)
    duelink.Graphics.Clear(0)
