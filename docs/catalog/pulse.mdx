## Graphical

![Pulse Display](pathname:///img/catalog/pulse-display.gif)


BrainPad Pulse has a 128×64 B&W graphical display that can be used for showing text, graphing sensor data as well as creating video games!

* Graphing sensor data
* Arcade-inspired games
* Text visuals
