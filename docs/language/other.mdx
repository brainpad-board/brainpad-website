# Other Languages

---

![And More](pathname:///img/other-languages.png)

Any system with serial port access can utilize DUELink modules. If [Excel](excel) can work with DUELink, anything else can!!

There are plenty of code samples to help you understand and port to whatever you like! See [DUELink Samples](https://github.com/ghi-electronics/duelink-samples) and [DUELink Libraries](https://github.com/ghi-electronics/duelink-libraries) repos.
