# TinyCLR

---

![TinyCLR OS](pathname:///img/tinyclr.png)

GHI Electronics TinyCLR OS is a tiny operating system that runs .NET C# on small circuits called [SITCore](https://www.ghielectronics.com/sitcore/). Just like the full [.NET](./dotnet) support, can extend [PCs & Laptops](../system/pc-laptop) with sensors and actuators, TinyCLR DUELink support extends SITCore devices with an entire [Catalog](../catalog/intro) of possibilities!
