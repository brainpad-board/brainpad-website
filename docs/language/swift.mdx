# Swift

![Swift Logo](pathname:///img/swift.png)

Swift can be used to give MacOS access to the physical world.

The support for SWIFT is incomplete and only provided as a reference for those interested in exploring the possibilities.

An example is found on the [DUELink Samples](https://github.com/ghi-electronics/duelink-samples/tree/main/Demo/MacOS) Repo.
