# Standalone

---

DUELink's [Scripting](./scripting) Language allow modules to run Standalone. The entire [Catalog](../catalog/intro) of modules can run Standalone. This is specifically important on [microcomputers](../catalog/microcomputers) such as [Ghizzy](../catalog/microcomputer/ghizzy.mdx) and [DueDuino](../catalog/microcomputer/dueduino).

![Ghizzy Eyes](pathname:///img/ghizzy-eyes.gif)

A Standalone module can also host/command a chain of [Daisylinked](./daisylink) modules.

![Daisylink](pathname:///img/powermin-daisylink.png)
